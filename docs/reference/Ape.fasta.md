# Save an Ape Instance as a Fasta File

Save an Ape Instance as a Fasta File

## Usage

``` r
Ape.fasta(ape, feature = NULL, out)
```

## Arguments

- feature:

  Name of feature to select when writing FASTA file. If null (default),
  the whole biostring will be saved as a fasta.

- out:

  Name of FASTA file to write
