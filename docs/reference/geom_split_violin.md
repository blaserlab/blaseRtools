# geom_split_violin

create a split violin geom

## Usage

``` r
geom_split_violin(
  mapping = NULL,
  data = NULL,
  stat = "ydensity",
  position = "identity",
  ...,
  draw_quantiles = NULL,
  trim = TRUE,
  scale = "area",
  na.rm = FALSE,
  show.legend = NA,
  inherit.aes = TRUE
)
```

## Arguments

- mapping:

  Set of aesthetitc mappings, Default: NULL

- data:

  Data to display, Default: NULL

- stat:

  Statistic to plot, Default: 'ydensity'

- position:

  Position, Default: 'identity'

- ...:

  extra arguments

- draw_quantiles:

  Quantiles to draw, Default: NULL

- trim:

  Trim ends?, Default: TRUE

- scale:

  Analogous to violin scale, Default: 'area'

- na.rm:

  Default: FALSE

- show.legend:

  Default: NA

- inherit.aes:

  Default: TRUE
